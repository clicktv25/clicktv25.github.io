Instrucciones de Instalación para Android

Sigue estos pasos para instalar ClickTV en tu dispositivo Android:

  1. Descarga el archivo APK de ClickTV desde el siguiente enlace: Descargar ClickTV para Android (APK)
  2. Una vez descargado, abre el archivo APK. Si te pide permisos para instalar aplicaciones de fuentes desconocidas, otórgalos en la configuración de tu dispositivo.
  3. Sigue las instrucciones en pantalla para completar la instalación.
  4. Una vez instalado, abre la aplicación ClickTV y disfruta.
